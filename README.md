# Rayleigh-Benard Convection 2D

PHYS 6260 Term Project

## Dependencies

This code is based on Taichi with a Python frontend.

```bash
# python 3.9
pip install taichi -U
pip install matplotlib
```

## Execution

```bash
python ./exec/rbc2d.py
```

Generate the video:

```powershell
./scripts/gen_video.ps1 ./exps/rbc T
```
